## Acid Burn
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Acid Burn</h2>
<p>This program creates self-replicating destructive code that targets the hardening of the device.</p>
<p>The decker makes a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Electronic Warfare)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Acid Burn</span> rating test. The target defends as normal. Hardening does reduce the number of net successes.</p>
<p>If successful, the hardening of the target is lowered by the number of net successes plus software rating divided in half, rounded down.</p>

## Alert Monitor
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Alert Monitor</h2>
<p>This software lets the decker know the current value of the alert monitor and the next expected response.</p>

## Analysis Locus
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Analysis Locus</h2>
<p>The decker makes a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Electronic Warfare)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Analysis Locus</span> rating test.</p>
<p>If she gets more successes than the target's hardening value, she knows and is dynamically updated by the target's cohesion status monitor value, and the attack and defense software loaded. These successes can be accumulated.</p>

## Corrupt IFF
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Corrupt IFF</h2>
<p>The decker makes a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Electronic Warfare)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Corrupt IFF</span> rating test.</p>
<p>She then selects a number of targets equal to her successes. Those targets are no longer identified as friendly to automated defenses. Drones, gun emplacements, and other automated targeting devices will now consider them valid targets.</p>

## Crack Encryption
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Crack Encryption</h2>
<p>This software allows you to access encrypted and secured files. You need to get a number of successes on a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Hacking)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Crack Encryption</span> rating test equal to six times the file security rating in order to access the file.</p>
<p>Once you have access, you may copy, control, delete, or take any other action you wish with the file. File security rating varies, from 1 for a public library site file to 6 for a classified military document. More advanced encryption reduces the number of dice the decker can use on a <span style="font-family:'Bruno Ace', sans-serif;">Crack Encryption</span>.</p>
<p><span style="font-family:'Bruno Ace', sans-serif;">Crack Encryption</span> loads into the I/O, so you can continue to make this test until all the successes are acquired necessary to access the file.</p>

## De-Rez
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">De-Rez</h2>
<p>This software allows you to disrupt network daemons and intrusion contact elimination software.</p>
<p>The decker opens up one of his I/O ports and then rolls a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Electronic Warfare)</span> + <span style="font-family:'Bruno Ace', sans-serif;">De-Rez</span> rating test. Hardening and defensive routines reduce these successes. For every remaining success, the network daemon or ICE takes 1d6 direct damage to their cohesion track.</p>

## Encrypt File
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Encrypt File</h2>
<p>This software allows a decker to make a file harder to access.</p>
<p>The hacker makes a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Hacking)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Encrypt File</span> rating test, and the first three successes increase the file security rating by 2 up to 6 for the <span style="font-family:'Bruno Ace', sans-serif;">Crack Encryption</span> program. Additional successes are added as a penalty to the tests of anyone attempting to crack the encryption. Encrypted programs can't be accessed, moved, used, or modified while encrypted. The person who encrypted the file can decrypt it for their own personal use without a test.</p>

## Ghost Protocol
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Ghost Protocol</h2>
<p>This software allows the decker to create a temporary false digital persona for an operation. During the Reconnaissance and Preparation framework a decker can replace one of their options with the <span style="font-family:'Bruno Ace', sans-serif;">Ghost Protocol</span>.</p>
<p>The decker makes a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Hacking)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Ghost Protocol</span> rating test. Every success creates a layer of protection for a fictional persona. This allows the decker (and her allies) to fake access, bypass security clearance, and otherwise appear as a legitimate person on site. The number of successes is the target number of successes that on-site security needs to discover the deception.</p>
<p>Characters in a brand also have false digital personas, a Ghost Rating. These can be enhanced by a hacker with a <span style="font-family:'Bruno Ace', sans-serif;">Ghost Protocol</span> program. A decker may take a sector action, using the brand's grid and use successes to increase a character's Ghost Rating. Successes can be accumulated across sector turns.</p>

## Shadow Protocols
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Shadow Protocols</h2>
<p>This software is designed to lower the alert level. It can be used actively to reduce the alert level, or loaded into the I/O stream to reduce all alert increases by its software rating.</p>
<p>Make a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Hacking)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Shadow Protocols</span> rating test as a complex action. Each success reduces the alert level by 1.</p>

## Device Control
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Device Control</h2>
<p>This software allows deckers to take control of devices and systems within range of their influence.</p>
<p>The decker must make a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Electronic Warfare)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Device Control</span> software rating test. They need a number of successes equal to twice a device's hardening. If they succeed, <span style="font-family:'Bruno Ace', sans-serif;">Device Control</span> goes into the I/O stream, and they gain control of the device and can use or deactivate any of its functions. This raises the alert value by the hardening of the device controlled.</p>
<p>Unlike hacking which can just turn devices on or off, this software allows complete control of the device as if she were the administrator.</p>

## Decoy
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Decoy</h2>
<p>The decker picks a target and loads the <span style="font-family:'Bruno Ace', sans-serif;">Decoy</span> software into his I/O stream.</p>
<p>As long as this is running, anyone firing on the target receives faulty information and has their targeting screwed up by the noise and holographic presentations only they can see and hear. These shadow signals and haze duplicates protect the target.</p>
<p>When fired upon, the target may use the program's rating as bonus dice added to all dodge tests.</p>

## Electric Strike
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Electric Strike</h2>
<p>This software causes an electrical discharge from nearby electrical equipment within the decker's influence field.</p>
<p>The decker makes a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Hacking)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Electric Strike</span> rating test. It does 1d6 direct damage to the target's stun condition track, plus an additional 1 point per success. This damage ignores armor. It cannot be dodged, but can be soaked.</p>
<p>The decker can distribute this damage to anyone in range, divided however they wish. If targeting machines, software agents, or synthetics, damage assigned is applied twice.</p>

## Emotional Influence
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Emotional Influence</h2>
<p>This software allows the decker to manipulate and alter the emotions of the targets in the field using subsonics, holograms, and soft musical clues.</p>
<p>Targets may resist by rolling as many dice as they wish from their <span style="font-family:'Bruno Ace', sans-serif;">Resolve</span> pool, versus the decker's <span style="font-family:'Bruno Ace', sans-serif;">Computer (Hacking)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Emotional Influence</span> rating test. The decker can evoke sadness, happiness, despair, or any other emotion she wishes. This can support other tests, providing a number of bonus dice equal to the rating of the <span style="font-family:'Bruno Ace', sans-serif;">Emotional Influence</span> software.</p>

## Hypnotic Projection
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Hypnotic Projection</h2>
<p>This software creates a distracting and hypnotic colorful strobe display.</p>
<p>Targets may resist by rolling as many dice as they wish from their <span style="font-family:'Bruno Ace', sans-serif;">Focus</span> pool, versus the decker's <span style="font-family:'Bruno Ace', sans-serif;">Computer (Electronic Warfare)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Hypnotic Projection</span> rating test.</p>
<p>Any who fail to achieve more successes than the decker stand in a stupor, fascinated and unable to act. On their turn, they may take a complex action to attempt to resist and break free. Any damage or someone jostling them with a simple action renders a target immunity to this effect.</p>

## Refraction Field
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Refraction Field</h2>
<p>This software allows the decker to manipulate a strong electromagnetic field. She can draw a straight line that intersects any 2 edges of her range of influence.</p>
<p>Make a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Electronic Warfare)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Refraction Field</span> software rating test. Lasers and other energy weapons fired through the barrier drop in power by 1 per success. The field does not impede physical movement or normal gunfire.</p>

## Situational Advantage
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Situational Advantage</h2>
<p>The decker uses this to distribute targeting information to allies in the radius. It alters the environment (lights, holograms, sounds) to support allies and harass enemies.</p>
<p>The decker selects a number of allies equal to the program's rating and loads the <span style="font-family:'Bruno Ace', sans-serif;">Situational Advantage</span> software into his I/O stream. All selected allies within the range of the decker's influence reroll 1's on all <span style="font-family:'Bruno Ace', sans-serif;">Brawn</span> and <span style="font-family:'Bruno Ace', sans-serif;">Finesse</span> pool skills.</p>

## Sonic Sickness
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Sonic Sickness</h2>
<p>The decker uses disruptive sonics to attempt to disable a number of targets equal to the rating of the software.</p>
<p>The decker makes a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Hacking)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Sonic Sickness</span> rating test. Targets on their turn must succeed at a <span style="font-family:'Bruno Ace', sans-serif;">Brawn</span> pool test with at least this many successes or they are limited to a single simple action on their turn. This lasts until they succeed twice in a row or the decker ends the program.</p>

## Targeted Disruption
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Targeted Disruption</h2>
<p>The decker uses this to manipulate sound waves, holograms, and more to befuddle a target.</p>
<p>The decker selects a target and loads <span style="font-family:'Bruno Ace', sans-serif;">Targeted Disruption</span> to his I/O stream. They make a <span style="font-family:'Bruno Ace', sans-serif;">Computer (Electronic Warfare)</span> + <span style="font-family:'Bruno Ace', sans-serif;">Targeted Disruption</span> rating test and count the successes.</p>
<p>The target makes a <span style="font-family:'Bruno Ace', sans-serif;">Resolve</span> pool test using as many dice as they wish and counts their successes. If the hacker gets more successes than the target, the target has 4 penalty dice on all tests. This lasts until the start of the decker's next turn.</p>
<p>Once loaded into the I/O stream, the decker may select a target and use <span style="font-family:'Bruno Ace', sans-serif;">Targeted Disruption</span> on them as a simple action. This software cannot affect more than one target at once.</p>

## Universal Translator
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Universal Translator</h2>
<p>The decker can use the access nodes to translate between any languages in range. Everyone will hear a translation on their internal gear.</p>
<p>If a target doesn't have any internal hardware to receive a translation, nearby sound projection apparati will broadcast it audibly to the area the unenhanced target is standing.</p>

## Vent Gas
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Vent Gas</h2>
<p>This allows the decker to access central temperature and climate control to vent gas and mist.</p>
<p>The decker can choose one 2 m (1") square per success and fill it with an opaque fog or mist. This obscures vision through the barrier but not movement. It provides total cover.</p>

## Vermin Call
<h2 style="font-family:'Bruno Ace', sans-serif; margin:0 0 0.5em 0;">Vermin Call</h2>
<p>The decker causes certain inaudible frequencies to surround a target. At the start of the next round, vermin swarm the target; silverfish, electrical ants, cockroaches, bedbugs, and more, depending on the environment, begin crawling and biting the target.</p>
<p>The target is unable to concentrate on any actions, preventing them from taking any complex actions while the effect lasts, and the decker does a direct 1 + 1/3 software rating in damage (round down) to the target's physical condition track at the start of each round. Targets in power armor, synthetics, spirits, and software agents are immune.</p>
